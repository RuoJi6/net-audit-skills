## 1. 鉴权状态说明

| 状态 | 含义 | 风险提示 |
| --- | --- | --- |
| 公开 | 明确允许匿名访问 | 需确认业务是否允许公开 |
| 受保护 | 有登录、角色/Policy 或资源级授权证据 | 无明显风险 |
| 仅认证 | 只确认登录态，未确认角色/资源归属 | 可能存在越权 |
| 无鉴权 | 未发现任何后端鉴权证据 | 高风险 |
| 不确定 | 证据不足或反编译/代码缺失 | 需人工或动态验证 |
